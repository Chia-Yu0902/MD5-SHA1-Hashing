# MD5_SHA-File-Integrity-check
## MD5簡介
> 全名為：Message-Digest Algorithm，5表示第五代
* 一種被廣泛使用的密碼雜湊函式，可以產生出一個128位元（16位元組）的雜湊值（hash value），用於確保資訊傳輸完整一致。

* 但MD5已經被證實無法預防「碰撞攻擊」，所以已遭禁用於SSL公開金鑰認證或是數位簽章等用途。
